# 🌸 Iris Flower Classification | Oasis Infobyte Data Science Internship (Task 1)

## 📌 Project Overview

This project is completed as **Task 1** of the **Oasis Infobyte Data Science Internship**. The objective is to build a Machine Learning model that accurately classifies Iris flowers into three different species based on their physical measurements.

The Iris dataset is one of the most well-known datasets in machine learning and is commonly used for classification problems.

---

## 🎯 Objective

To develop a machine learning classification model that predicts the species of an Iris flower using the following features:

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

The target classes are:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

---

## 📂 Dataset

The project uses the **Iris Dataset**, which contains:

* **150 samples**
* **4 numerical features**
* **3 flower species**
